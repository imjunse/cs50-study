# Harvard CS50 (2026) 스터디 사이트

> 하버드 코딩 입문 강의, CS50x 2026을 함께 공부하기 위한 웹 스터디 가이드입니다.

🔗 **사이트 바로가기**: [imjunse.github.io/cs50-study](https://imjunse.github.io/cs50-study/)

## 이 사이트는 뭔가요?

[CS50](https://cs50.harvard.edu/x/)은 하버드 대학교에서 제공하는 컴퓨터 과학 입문 강의로, 전 세계에서 가장 인기 있는 프로그래밍 강좌 중 하나입니다. 이 사이트는 CS50x 2026 강의를 들으면서 **챕터별 노트 확인, 퀴즈 풀기, 진도 체크**를 한 곳에서 할 수 있도록 만든 스터디 도우미입니다.

## 주요 기능

- 📺 **YouTube 강의 임베드** — 챕터를 클릭하면 해당 시점으로 바로 이동
- 📝 **챕터별 학습 노트** — 핵심 개념을 한눈에 정리
- ✅ **퀴즈** — 각 챕터마다 이해도를 확인하는 객관식 퀴즈
- 📊 **진도 관리** — 완료한 챕터를 체크하면 진행률이 자동 저장
- 🌙 **다크 모드** — 눈이 편한 다크 테마 지원
- 🆓 **자유로운 학습 순서** — 원하는 주차부터 자유롭게 시작 가능
